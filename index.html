• git config --global user.name blabla - ввести своє ім'я для підключення github
  • git config --global user.email blabal@bla ввести свою електронну адресу для підключення github
  • git init - ініціювати створення локального репозиторію
  • git status - переглянути статут файлів на локальному репозиторії
  • git add - додати файли в stage
  • git commit - закомітити файли(створити зліпок файла)
  • git log - переглянути історію комітів(можна витягнути ід конкретного коміта і повернутись до відповідної версії)
  • git checkout - для переключання гілок репозиторію
  • git branch
  • git push
  • git pull
  • git merge
  • git clone
  • git ignore
